![Screenshot](https://imgur.com/Q8BputN.png)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# lint the Typescript
npm run lint

# lint the Sass
npm run sasslint

# run the tests
npm test

# run the mutation tests
npm stryker

# build for production with minification
npm run build

# clean the production build
npm run clean
```
